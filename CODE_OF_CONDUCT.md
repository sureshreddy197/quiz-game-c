# Code of Conduct

## Our Pledge

We want this project to be welcoming, respectful, and useful for learners.

## Our Standards

Examples of positive behavior:

- Being respectful in discussions and reviews
- Giving constructive feedback
- Helping beginners without gatekeeping
- Keeping pull requests focused and readable

Examples of unacceptable behavior:

- Harassment or personal attacks
- Abusive language
- Deliberate disruption of collaboration
- Dismissing contributors in bad faith

## Enforcement

Project maintainers may remove or reject contributions that violate this code of conduct.
