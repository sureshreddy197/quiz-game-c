# Security Policy

## Supported Versions

This project is a learning-focused console application. Security fixes will be applied to the latest version on the default branch.

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |
| older   | No        |

## Reporting a Vulnerability

Please do **not** open a public issue for security problems.

Instead:

1. Email the maintainer with a clear description of the issue.
2. Include reproduction steps, affected files, and any proof of concept if available.
3. Allow reasonable time for review and remediation before public disclosure.

## Scope

Examples of issues that are worth reporting:

- Unsafe file handling
- Memory corruption or crashes caused by malformed input
- Privilege or path traversal problems if external file inputs are added later

Examples that are usually out of scope for this project:

- Cosmetic terminal rendering bugs
- Missing gameplay features
- General C language warnings without a practical exploit path
